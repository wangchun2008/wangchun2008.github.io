个人简介

我有 10 年的 IT 系统集成运维经验。参与多个保险、金融行业集成运维项目。善于快速学习新技术，在重压下高效完成项目建设。既懂得传统行业 IT 项目集成风格，又了解互联网行业开源技术。现任职安华亿能医疗影像科技（北京）有限公司系统工程师，负责公司心电监测平台、颈动脉筛查平台等建设与维护，在节约成本的前提下，建设高效、安全的业务平台，对外提供高质量服务。目标是成为运维技术专家，做个与时俱进的技术人。

工作经历

安华亿能医疗影像科技（北京）有限公司 （ 2015 年 4 月 ~ 今 ）

期间任职系统工程师，负责心电平台、颈动脉筛查平台的搭建和维护工作。

数据库运维

负责数据库（MySQL、Redis）建设与维护。MySQL 高可用采用主主架构，Redis 采用主从架构。在任职期间，没有发生过数据服务中断事件。可以及时响应业务方面对数据的各种需求。

  • 根据需求设计、迭代更新心电系统、Portal 系统、筛查系统等数据库。
  • 从硬件选型、系统、软件配置参数、SQL 查询、数据库结构、索引方面对数据库进行优化。
  • 制定数据库备份策略（全备、增量），检查数据备份有效性。
  • 对公司的开发、测试、生产环境、客户环境数据库进行维护，包括但不限于数据修改，数据的导入导出等操作。

系统运维

负责服务器硬件、Web 应用、内部应用的建设与维护。

  • 对服务器硬件进行巡检、资源扩容等。
  • 建设 Zabbix 监控平台，对服务器硬件、操作系统、Web 应用、数据库高可用等指标进行监控与预警。
  • 建设 Gitlab 平台统一管理所有项目代码。
  • 使用 Gitlab + Ansible + Jenkins 方式实现代码的自动部署及接口测试，节省部署时间，提高了交付效率。
  • 对 Web 应用配置启用 HTTPS 协议进行访问。

其它工作

  • 创建维护运维项目文档。
  • 使用 Docker 搭建 Redmine 系统。
  • 培训：Git 命令、Gitlab 使用规范。
  • 使用 Shell 编写心电、颈动脉环境一键部署脚本。
  • 对现有系统迁移到公有云（阿里云、腾讯云）进行可行性评估，包括技术差异、架构变更、投入估算等方面。

北京新锐互动商业网络有限公司 （ 2012 年 7 月 ~ 2015 年 4 月 ）

期间任职系统工程师，将近 2 年时间在中央国债登记结算有限责任公司驻场，其它时间进行短期的系统建设工作。

中债信息网建设运维

  • 进行 F5、Web 服务器、应用服务器、数据库服务器、带库、盘阵的巡检维护。
  • 安装配置维护 F5 + IHS + WebSphere Application Server（WAS）+ Oracle RAC 集群环境。
  • 安装配置维护 Real User Experience Insight 系统。（ORACLE 公司的真实用户体验系统）。

IBM PORTAL 系统环境建设

  • 中国人寿内网 Portal 6.5 的集群安装配置工作。在一个星期内，通过红皮书进行突击学习，完成了 Oracle DB + Portal + LDAP 的集成工作。
  • 中国银行 MISP4 期项目，包括 IHS 集群、WAS8 集群、Portal 8 集群、安装配置工作。

其它项目

  • 建设大连银行外网门户系统，包括 Web 应用、访问量统计应用、WAS 服务器、防篡改服务器的安装配置测试工作。
  • 建设东方CJ、歌华有线、中银保险、财政部、中债的 RUEI 系统。
  • 进行徽商银行、齐鲁银行、国盛证券网站巡检、应用升级、服务器硬件升级工作。
  • 建设金谷国际信托外网门户系统，包括 Web 应用、访问量统计应用、WAS 服务器、防篡改服务器的安装配置测试工作。

中科软科技股份有限公司 （ 2010 年 9 月 ~ 2012 年 7 月 ）

期间本人任职系统工程师。在中国保险行业协会车险信息平台项目组驻场，负责所有 WAS 平台的建设维护工作。平台包括 134 台应用服务器，配置有 300 多个 WAS 节点，30 个应用集群。日常工作是对 HP 刀片服务器、Redhat 系统、WAS 集群巡检，对硬件、软件故障进行处理。

北京中科大洋科技发展股份有限公司 （ 2008 年 7 月 ~ 2010 年 7 月 ）

期间本人任职运维工程师，在中央电视台音像资料馆驻场。2 年中与同事一起维护资料馆媒体资源管理系统的稳定运行。 日常工作是对 Oracle 数据库，Windows 平台、Unix 平台服务器，EMC 盘阵、HP 盘阵、UIT 盘阵、日立盘阵，数据流磁带库等进行维护与保养。
